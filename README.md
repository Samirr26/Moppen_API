# Samir's Jokes API
An API to retrieve Jokes

## Use cases

### Receive random joke
GET: /random

### Receive list of jokes based on subject
GET: /subject/{subject} <br />
e.g. : /subject/sports

### Receive list of jokes based on author
GET: /subject/{author} <br />
e.g. : /subject/samir
